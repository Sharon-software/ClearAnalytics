ClearAnalytics

📊 AI-Powered Data Insights Dashboard

An AI-driven analytics platform that automatically transforms raw CSV/Excel data into interactive dashboards, business insights, and executive reports using OpenAI API.

It works like a lightweight AI alternative to Power BI/Tableau, but with natural language explanations and automated insight generation.

🚀 Features
📁 Data Upload
Upload CSV and Excel files (.csv, .xlsx)
Large dataset support
Instant data preview in table format
Multiple file support per user
🧠 AI Data Analysis (Core Engine)

Powered by OpenAI API to automatically generate insights:

Key Performance Indicators (KPIs)
Trend detection
Pattern recognition
Correlation analysis
Anomaly detection
Predictive insights (where possible)
Natural language data summary
📄 AI Executive Report Generator

Automatically generates a structured business report:

Executive Summary (plain English)
KPI Analysis
Trend Analysis
Anomaly Report
Key Insights (bullet points)
Actionable Recommendations
📊 Interactive Dashboard

Power BI-style analytics interface:

KPI metric cards
Line charts (trends over time)
Bar charts (comparisons)
Pie charts (distribution analysis)
Filters (date, category, variables)
💬 AI Chat Assistant (Data Copilot)

Ask questions about your data in natural language:

“Why did sales drop in March?”
“What is the top-performing region?”
“Show anomalies in revenue”
“What trends should I care about?”
📤 Export System
Export insights report as PDF
Download charts as images
Export full dashboard summary
